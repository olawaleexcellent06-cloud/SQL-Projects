# SQL-Projects
SQL data analysis projects focused on querying, cleaning, and analyzing structured datasets using joins, aggregations, and filtering techniques to extract business insights.

This repository contains SQL-based data analysis projects that demonstrate the ability to extract insights from structured databases using queries, joins, and aggregations.
The focus is on business analysis, customer behavior, product performance, and data-driven reporting.

Tools Used
SQL (MySQL / PostgreSQL syntax)
SELECT, WHERE, ORDER BY
JOIN
GROUP BY & Aggregations
COUNT, SUM, AVG functions
